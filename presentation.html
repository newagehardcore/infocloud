Whats Going On

Real-time Interactive News Tag Cloud Application

React TypeScript Three.js NLP Responsive Design

Project Overview

"Whats Going On" is a real-time news aggregator visualization application that displays current news as an interactive 3D tag cloud. The application visualizes news topics as words sized by frequency, colored by political leaning, and allows users to explore stories by clicking on terms. It includes a time-travel feature that lets users rewind to see how news evolved throughout the day.

This innovative approach to news browsing helps users quickly identify trending topics, understand media bias, and explore news stories in an engaging, interactive way.

Key Features

3D Tag Cloud Visualization

Interactive 3D cloud where words represent news topics, sized by frequency and colored by political bias.

Real-time Updates

News continuously flows into the visualization with new words animating into view and existing words dynamically resizing.

Interactive Exploration

Click on words to reveal related news stories, source information, and links to original articles.

Category Filtering

Filter news by categories: World, US, Tech, Business, Entertainment, Sports, Health, Science.

Time Machine

Rewind to see how news evolved throughout the day with clear visual indicators for historical vs. current data.

Responsive Design

Optimized for both desktop and mobile devices with adaptive rendering based on device capabilities.

Technical Implementation

Architecture

The application is built using React with TypeScript for the frontend framework, providing type safety and improved developer experience. Three.js is used for the 3D visualization, allowing for an immersive and interactive tag cloud. The application follows a component-based architecture with clear separation of concerns.

Data Processing

News data is fetched from a News API and processed using natural language processing techniques to extract key terms. The application analyzes and categorizes media bias, storing historical snapshots for the time machine feature. Updates occur every 5-10 minutes to maintain a real-time feel.

Performance Optimization

The application includes several performance optimizations:

  • Adaptive rendering based on device capabilities
  • Throttling and debouncing for expensive operations
  • Simplified view for mobile and low-power devices
  • Frame rate monitoring with quality adjustments
  • Memoization for caching expensive calculations

Responsive Design

The application is fully responsive, providing an optimized experience across devices:

  • Desktop: Full 3D visualization with interactive features
  • Mobile: Simplified 2D tag cloud with touch-friendly interactions
  • Low-power devices: Automatically adjusted rendering quality
